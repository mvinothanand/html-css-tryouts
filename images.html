Image set with max-width : 100% and max-height: 100%doesn't overflow

Image with intrinsic size overflows the box.

Image sized with object fit: cover. Maintains the aspect-ratio of the image. However, some portions of the image get cropped.

Note: Using object-fit requires height and widht of the image to be defined.

Image sized with object-fit: contain. aspect-ratio is maintained without cropping. Might leave empty space.

Image sized using object-fit: fill. aspect-ratio is not maintained, but the full image is within the box.
