CSS Flexbox

Flexbox is a one dimensional layout method, for arranging items in rows and columns. Items flex (stretch/shrink) to fit in the available space.

Flex Direction

Value of flex-direction property decides the direction in which the flex items are laid out.

flex-direction: row

row is the default flex direction. The items are laid out in the same direction as the writing mode for the language.

Sizing the flex items

The flex CSS property shall be set on the flex items to control the sizing. It is a shorthand of flex-grow, flex-shrink and flex-basis properties.

flex-grow: A unitless value to tell flexbox container how much proportion of the available space the flex item should occupy.

flex-shrink: An unitless proportion value which defines how much a flex item can shrink to avoid overflow.

flex-basis: Minimum size of the flex item in measuring units.

Aligning Flex Items

align-items: To align items along the cross-axis. Applied on the flex-container.

align-self: To control alignment in the cross-axis applied on the flex item.

justify-content: To control alignment in the main-axis applied on the flex container.

Aligning Flex Items in the cross-axis

align-items: stretch;

JavaScript

React

align-items property has a default value of stretch.

stretch adjusts all the flex items to be of same size in the cross-axis. The size of each of the flex item is increased to be same as that of the largest item based on the intrinsic size.
